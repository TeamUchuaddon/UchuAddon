# UchuAddon

**UchuAddon** は、**Nebula on the Ship** 向けに作られたアドオンです。  
新しい役職やゲーム要素を追加し、プレイ体験を拡張します。

---

## 概要

- **種別**：Addon
- **ベースMOD**：Nebula on the Ship

---

## 移植・二次利用に関するガイドライン

本アドオン（UchuAddon）の移植および二次利用については、以下のガイドラインを遵守してください。

### 1. 役職アイディアおよび仕様について
**【移植の許可】** UchuAddonが独自に考案した役職のアイディア・ゲーム性・ロジックに関しては、著作権を主張いたしません。他環境への移植や、改変した役職の作成は自由に行っていただいて構いません。

> [!IMPORTANT]
> ただし、他MOD由来の役職については本項は適用されません。詳細は「3. 他MOD由来の役職について」をご確認ください。

### 2. 独自リソースの利用禁止
**【禁止事項】** アドオン内に含まれる以下のリソースを、そのまま（あるいは改変して）抜き出し、他MODやプロジェクトへ移植・利用することは**一切禁止**いたします。

* ボタンイラスト
* 画像・テクスチャ
* エフェクト
* その他各種リソース

### 3. 他MOD由来の役職について
**【遵守事項】** 本アドオンに実装されている他MOD由来の役職に関しては、それぞれの元MODが定める規約およびガイドラインに従ってください。

**【免責事項】** 他MODの役職移植に関するトラブルや権利侵害について、本アドオン制作側は一切の責任を負いません。
